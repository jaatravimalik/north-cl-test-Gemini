# NorthIndia Connect

**Apna Network, Apni Pehchaan.** 
A full-stack community platform built for North Indian professionals and businesses to connect, share updates, and discover local services.

## Tech Stack
- **Frontend**: React.js, Vite, React Router, Axios, Context API
- **Backend**: NestJS, TypeORM, PostgreSQL, Passport JWT, Multer
- **Design**: Vanilla CSS with a custom Saffron (#FF6B1A), Navy (#0F1C2E), and Cream (#FFF8F0) palette

## Features
1. **User Authentication**: Secure JWT-based login and registration with access and refresh tokens.
2. **Professional Profiles**: LinkedIn-style user profiles with avatars, cover photos, experience, education, and skills.
3. **Community Feed**: Post updates with images, like posts, and engage in comment threads.
4. **Business Directory**: Browse local businesses by category, search by name, and view detailed listings.
5. **Ratings & Reviews**: Leave star ratings and text reviews for businesses in the directory.

## Getting Started

### Prerequisites
- Node.js (v18+)
- PostgreSQL (running locally or remotely)

### 1. Database Setup
Ensure PostgreSQL is running. Create a new database named `northindia_connect`:
```sql
CREATE DATABASE northindia_connect;
```

### 2. Backend Setup
1. Navigate to the backend directory:
   ```bash
   cd backend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Copy `.env.example` to `.env` and update the database credentials if necessary:
   ```bash
   cp .env.example .env
   ```
4. Start the NestJS server:
   ```bash
   npm run start:dev
   ```
   *The API will run on `http://localhost:3000`.*

### 3. Frontend Setup
1. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Copy `.env.example` to `.env` (it contains the Vite proxy target):
   ```bash
   cp .env.example .env
   ```
4. Start the Vite development server:
   ```bash
   npm run dev
   ```
   *The app will run on `http://localhost:5173`.*

## Directory Structure
- `/backend` - NestJS API application, TypeORM entities, and modules.
- `/frontend` - React single-page application and UI components.
- `/uploads` - (Generated by backend) Stores user-uploaded avatars, covers, and post images.
